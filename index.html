Prework Study Guide

HTML

  • The <head> contains information about the webpage, and can contain other elemnts:
    • The <meta> element defines metadata such as charcter set, description, keywords, author, and viewport.
    • The <style> element directs the CSS attributes or links a CSS file.
    • The <base> element denotes the base URL for the website.
  • The <body> tag represents the visible content shown to the user.

CSS

  • Each element on a wepage is essentially a box - Box Model
  • Margin - indicates how much space we want around the outside of am element
  • Padding - indicates how much space we want around the content on the inside of the border of an element

Git

  • $ git clone git@github.com:Jacob-Jeffries/prework.git - clone a repo locally
  • $ git status - provides the current status of our repo
  • $ git branch - shows the current branches
  • $ git checkout -b feature/add-html - this creates the feature branch: feature/add-html
  • $ git pull origin main - fetch and merge any commits from the tracking remote branch
